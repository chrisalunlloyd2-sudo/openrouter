## Setup Instructions
### Windows Setup
1. Install Python 3.10+ from python.org
2. Open PowerShell
3. Run: `pip install -r requirements.txt`
4. Execute: `python src/main.py`

### Android Setup (Termux)
1. Install Termux
2. `pkg install python git`
3. `pip install -r requirements.txt`
4. `python src/main.py`

## Node Descriptions
### Director Node
The Director Node is responsible for intercepting user input, hashing it for token efficiency caching, and routing it to the Executor Node.

### Executor Node
The Executor Node is responsible for parsing the output from the Director Node using Python Regex.

### Syphon Node
The Syphon Node is responsible for pushing the extracted codebase to GitHub.
